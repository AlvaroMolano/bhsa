Introduction¶

This is the key to the meaning of the features of the BHSA dataset.

We organize the features in several groups, roughly analogous to the types of objects we have:

  • grid
  • sectional
  • word
  • lexeme
  • linguistic
  • relationships
  • generic

Grid features¶

---|---|--- otype | node type | book verse clause phrase word oslots | slot containment | 1 1-11 2010-2015,2020-2030 otext | textapi | no data, only specifications

Sectional features¶

---|---|--- book | name of Bible book | Genesis Psalmi Amos chapter | number of chapter within book | 3 verse | number of verse within chapter | 4 label | passage indicator | AMOS 03,04 label | key for part within verse | A B C
